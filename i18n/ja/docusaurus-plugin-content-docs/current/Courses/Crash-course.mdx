## これはどういうことですか？

👩‍💻 業界のデジタル化の流れの中で、**プログラミングの重要性はますます高まっています**。一見とっつきにくい分野ですが、とても楽しく、ビジネスの可能性が広がる分野です。

👩‍🏫 しかし、プログラミングの世界はあまりにも広大で、どこから手をつけていいのか判断に迷うところです。そこで、この講座を作りました。ここでは、**ゼロからプログラミングする方法と、IFC.js**でBIMアプリケーションを作成するための基礎を学ぶことができます。

<IfcCard>
BIMプログラミングにより、独自のツールを作成し、プロセスを自動化し、新しいビジネスを創造し、より良い仕事に就くための資格を得ることができるようになります。
</IfcCard>
